health
======

![build](https://travis-ci.org/elkaadka/health.svg?branch=master)


Checks the health of the services : Mysql and Redis

add the following parameters in yout parameters.yml 

(replace ... with the correct value) :

    parameters:
        secret: ...
        database_host: ...
        database_port: ...
        database_name: ...
        database_user: ...
        database_password: ...
        redis_host: ...
        redis_port: ...
